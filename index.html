FAQs - Frequently Asked Questions (Blockchain Cousera)

1. "truffle init" command is not working.

Solution:
1. Try git command: git clone https://github.com/truffle-box/bare-box/

2. rename bare-box folder to Ballot1: mv bare-box Ballot1

3. Open Migrations.sol in Gedit text editor

4. Change *pragma solidity >=0.4.21 <6.0.0;* to *pragma solidity ^0.4.17;*

5. Change *constructor() public* to *function Migrations() public*
